# Android Notes

This module mirrors the architecture in the root docs:

- `core/` contains transport and data models.
- `features/` contains user-facing remote experiences.
- `ui/` contains themes and reusable controls.

The pair-URI, mDNS discovery, WebSocket transport, and agent API flow are live. The remaining Android-side work is around QR camera scanning, persistence, and richer remote authoring.
