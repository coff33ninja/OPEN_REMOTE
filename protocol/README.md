# Protocol Assets

This directory holds the stable, implementation-neutral definitions for OpenRemote command and layout payloads.

- [`commands.md`](/j:/SCRIPTS/OPEN_REMOTE/protocol/commands.md): command envelope and builtin command catalog.
- [`remote-layout.schema.json`](/j:/SCRIPTS/OPEN_REMOTE/protocol/remote-layout.schema.json): starter JSON schema for renderable remotes.
- [`command-envelope.schema.json`](/j:/SCRIPTS/OPEN_REMOTE/protocol/command-envelope.schema.json): starter JSON schema for agent commands.
